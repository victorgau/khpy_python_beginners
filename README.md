## 課程內容：

這個課程主要是想要讓初學者藉由實際動手來熟悉Python程式的基本語法，如函式跟迴圈等。

### 適合對象：

小五以上稍微了解英文及電腦操作的朋友，最好具備座標系、角度及數學函數的基礎概念。
